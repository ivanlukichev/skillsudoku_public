## What It Is

SkillSudoku is a lightweight Sudoku website built for fast play, clean UX, and strong organic growth. The puzzle is visible on the first screen, gameplay runs without reloads, and the structure expands beyond a single game page into a full guide and intent-based content hub.

This public repository exists as a simple GitHub landing page for the project: one place to link the live site, show the core experience, and make the product easy to understand in a few seconds.

## Why It Feels Different

- Sudoku stays above the fold instead of hiding behind a banner.
- The interface is fast, touch-friendly, and intentionally uncluttered.
- Easy, medium, hard, and daily modes all share one consistent experience.
- Articles support both new players and long-tail search intent.
- The site is built with plain HTML, CSS, and vanilla JS for speed.

## Guide Topics

The guide is designed as a real content layer, not filler. It covers:

- how to play Sudoku
- practical tips and basic strategies
- common mistakes and why players get stuck
- Sudoku history and global spread
- difficulty, clue counts, and puzzle generation
- brain, focus, and habit-related questions
- championships, records, and Sudoku variations

Start here: [SkillSudoku Guide](https://skillsudoku.com/guide/)

## Project Snapshot

- Topic: classic Sudoku
- Stack: HTML, CSS, vanilla JavaScript
- Modes: easy, medium, hard, daily
- UX goal: instant play, mobile comfort, no reload gameplay
- SEO goal: clean structure, unique pages, internal linking, article clusters
